ASTEROID DETECTOR

ASTRO-CORE Observation System

A high-fidelity Android application prototype for tracking Near-Earth Objects (NEOs). Designed using Jetpack Compose and Google Stitch exports, built as a tribute to World Asteroid Day to promote asteroid awareness and STEM education.

  • Target HUD: Visual telescopic grid overlays and radar triangulation sweeps.
  • Sensors Uplink: Configurable noise reduction, exposure levels, and signal calibration.
  • Secure Vault: Double-encrypted reports portal for local astronomical observatories.
Target Environment Android Mobile HUD
Framework Stack Kotlin / Jetpack Compose

ISS CREW WEARABLE

NASA Smartwatch UI Dashboard

A glanceable Wear OS system interface tailored for astronauts aboard the International Space Station. Features OLED-friendly low-power styling, warning protocols, timers, and ground communications.

  • Glanceable Schedule: Orbit timeline monitoring with mission-critical alarms.
  • Caution & Warning System: Integrated warning telemetry indicating airlocks, pressure, and thermal health.
  • Interactive Simulator: Embedded Wear OS hardware emulator directly below.
Target Environment Wear OS (Google Pixel Watch)
Simulation Framework HTML5 / Vanilla CSS / JS Webview
